# Search books

## Установка

```bash
git clone https://github.com/glebgubckin/search-books
```
```bash
cd search-books
```
```bash
npm run build
npm run preview
```
Приложение будет запущено по адресу *http://localhost:3000*

## Docker

```bash
docker build -t search-books .
docker run -dp 127.0.0.1:3000:80 search-books
```
